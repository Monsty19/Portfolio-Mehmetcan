— API MCP

Le protocole MCP dans les API

Veille sur l'intégration du Model Context Protocol (MCP) dans les API modernes. MCP est une couche standardisée qui se pose au-dessus des API REST existantes pour les rendre consommables par les agents IA : découverte dynamique des outils, session persistante, interface unifiée — là où chaque API classique demandait une intégration manuelle spécifique.
